# 🎨 Design Decisions

The application was designed with scalability and maintainability in mind.

### Component-Based Architecture

Each section is developed as an independent reusable component.

Example:

- Hero
- HeroContent
- HeroBoard
- HeroButton
- HeroBackground
- FloatingPiece
- ChessBoard

This separation keeps the code clean and easy to maintain.

---

### Responsive Design

Tailwind CSS responsive utilities were used to ensure the interface works smoothly across:

- Mobile
- Tablet
- Desktop
- Large Screens

Typography, spacing, buttons, and layout adapt according to screen size.

---

### Animations

Framer Motion is used to create:

- Hero entrance animation
- Chessboard animation
- Floating chess pieces
- Interactive Play button
- Hover effects

Animations were kept lightweight to maintain performance.

---

### Chess Implementation

Chess functionality is powered by:

- Chess.js
- React Chessboard

Features include:

- Autoplay
- Replay
- Move Highlighting
- Move Arrows
- Board Animations
